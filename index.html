Charleston Lab

A practice & study guide for the American Mahjongg Charleston (2026 NMJL card).

  • Control Panel (v3)
    Six-zone study panel — direction, portfolio, pass builder, information ledger, replay reveal.
  • Decision Quality (v2)
    EV-loss grading of your pass choices.
  • Hand Rater (v1)
    Static starting-hand and direction rater.
  • Mobile PWA
    Installable, offline-capable mobile shell.
Source on GitHub. Probabilities are calibrated to a greedy rollout policy; point values are illustrative. See README / CLAUDE.md.
